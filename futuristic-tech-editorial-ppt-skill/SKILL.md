# Futuristic Tech Editorial PPT Skill

Use this skill when a presentation should feel like a premium technology magazine: editorial, precise, asymmetric, data-aware, and clean enough for executive or technical audiences.

## Style Intent

Create high-end editorial PPT slides with a futuristic tech magazine aesthetic.

- Background: ultra clean white, with a subtle grid system.
- Typography: bold modern sans-serif headings with strong hierarchy.
- Composition: large negative space, asymmetric structure, clear scan path.
- Visual language: thin geometric rules, data-inspired modules, interface-like framing.
- Finish: flat, sharp, professional, no shadows.
- Aspect ratio: 16:9 unless the user explicitly asks otherwise.

## Required Palette

- White: `#FFFFFF`
- Electric Blue: `#2F6BFF`
- Graphite Gray: `#2B2B2B`

Utility colors may be derived from the required palette only when needed for grid lines, dividers, or secondary labels:

- Grid line: `#E6EAF2`
- Secondary text: `#6B6B6B`
- Pale panel tint: `#F7F9FC`

## Non-Negotiables

- Use `#FFFFFF` as the dominant canvas.
- Use `#2F6BFF` sparingly as signal, not decoration.
- Use `#2B2B2B` for primary text and major structural elements.
- Keep slides flat: no shadows, no glow, no glassmorphism, no gradients, no 3D.
- Do not use decorative blobs, bokeh, clay objects, paper texture, or soft rounded lifestyle styling.
- Prefer thin lines, grid fragments, numeric labels, tags, coordinates, timelines, and data modules.
- Preserve wide margins and negative space.
- Build asymmetric layouts with deliberate alignment to a grid.
- Keep every slide sharp, editorial, and business-credible.

## Workflow

1. Read the source document and extract only the core narrative.
2. Choose one dominant idea per slide.
3. Build each slide from a clear editorial grid.
4. Use sparse electric blue accents to highlight the main signal.
5. Validate each slide against `references/qa-checklist.md`.
